# ALMA HEBRAL APPI

Este proyecto fue creado para entender y aplicar react js en mi e-commers

### Instalación
Fue creada con Node v22.19.0


### Librerias 
[React Icons](https://react-icons.github.io/react-icons/): utilizada para darle estilo al carrito.
[Sweetalert2](https://sweetalert2.github.io/): alertas de error.
[React Router Dom](https://reactrouter.com/): utilizada para implementar la navegación por rutas.
[Firebase](https://firebase.google.com/?hl=es-419): utilizada como base datos.
